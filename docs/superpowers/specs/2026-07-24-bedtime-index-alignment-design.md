# 취침 시간대 인덱스 정렬 설계

## 목표

취침 시간대 슬라이더의 다섯 시간대 설명을 각 숫자 아래 동일한 높이에 정렬한다.

## 원인

현재 2~4의 시간대 설명은 `indexLabels` 행에 렌더링되지만, 1과 5의 설명은 `leftDescription`과 `rightDescription`을 통해 별도의 아래쪽 행에 렌더링된다. 서로 다른 행을 사용하므로 수직 위치가 일치하지 않는다.

## 변경

`SurveySleep`의 취침 시간대 `Slider`에서 `leftDescription`과 `rightDescription`을 제거하고 다음 설명을 모두 `indexLabels`로 전달한다.

- 1: `10시 이전`
- 2: `10시~11시`
- 3: `11시~12시`
- 4: `12시~1시`
- 5: `새벽 1시 이후`

공통 `Slider` 컴포넌트와 다른 설문 슬라이더는 변경하지 않는다.

## 검증

- 기존 취침 시간대 매핑 테스트를 1~5 전체 매핑을 요구하도록 변경한다.
- 테스트를 먼저 실패시킨 뒤 `SurveySleep`의 props를 변경한다.
- 관련 테스트와 프로덕션 빌드를 실행한다.
