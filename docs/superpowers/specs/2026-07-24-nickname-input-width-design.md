# 닉네임 입력 너비 디자인

## 목표

닉네임 입력 영역이 가로 전체를 차지하지 않도록 8자 제한과 내부 주사위 버튼에 충분한 너비로 축소한다.

## 설계

`SurveyIntroduce`에서 닉네임 `InlineInput`만 `w-48 max-w-full` 래퍼로 감싼다.

- `w-48`은 한글 8자, 입력 왼쪽 여백, 오른쪽 주사위 버튼 영역을 수용한다.
- `max-w-full`은 좁은 화면에서 부모 영역을 넘지 않게 한다.
- `InlineInput` 자체 API와 다른 사용처의 전체 너비 동작은 변경하지 않는다.
- 주사위 버튼 위치, 닉네임 생성, 8자 제한, 접근성 라벨은 유지한다.

## 검증

정적 계약 테스트로 닉네임 입력에만 `w-48 max-w-full` 래퍼가 적용되는지 확인한다. 전체 테스트, ESLint, Vite 프로덕션 빌드를 실행한다.
