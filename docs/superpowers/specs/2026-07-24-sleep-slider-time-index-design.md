# 취침 시간대 슬라이더 인덱스 설계

## 목표

`SurveySleep`의 취침 시간대 슬라이더에서 숫자 2, 3, 4 아래에 대응하는 시간 범위를 표시한다.

## 컴포넌트

- `Slider`에 선택적 `indexLabels` 객체 속성을 추가한다.
- 키는 슬라이더 숫자, 값은 해당 숫자 아래에 표시할 문자열이다.
- 숫자 행 아래에 동일한 눈금 좌표를 사용하는 별도 인덱스 행을 렌더링한다.
- 값이 없는 눈금에는 빈 공간만 유지한다.
- 인덱스는 `text-[10px]`, `whitespace-nowrap`, 보조 텍스트 색상을 사용한다.
- `indexLabels`가 없으면 인덱스 행 자체를 렌더링하지 않아 기존 슬라이더 간격을 유지한다.

## SurveySleep 적용

- 2: `10시~11시`
- 3: `11시~12시`
- 4: `12시~1시`
- 세 문구는 기본적으로 숫자 아래에 표시한다.
- 모바일 눈금 간격과 문구 폭상 겹침 가능성이 낮으므로 교차 배치는 사용하지 않는다.

## 테스트

- Slider가 선택적 `indexLabels`를 눈금 좌표에 맞춰 렌더링하는 구조인지 검사한다.
- SurveySleep이 정확한 2·3·4 매핑을 전달하는지 검사한다.
- 기존 설문 레이아웃 테스트, 린트, 프로덕션 빌드를 실행한다.
