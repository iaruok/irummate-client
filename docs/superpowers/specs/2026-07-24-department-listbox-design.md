# 학부·학과 커스텀 리스트박스 설계

## 목표

UserDetails의 학부·학과 항목을 한국어 오름차순으로 표시하고 프로젝트의 입력 UI와 일관된 커스텀 리스트박스로 제공한다.

## 동작

- 전달받은 `items`는 변경하지 않고 `ko-KR` 기준으로 복사·정렬한다.
- 트리거 클릭, Enter, Space, ArrowDown, ArrowUp으로 목록을 연다.
- 열린 목록에서는 방향키로 항목을 이동하고 Enter로 선택하며 Escape와 바깥 클릭으로 닫는다.
- 선택 후 기존 `onChange(value)` 계약을 유지한다.

## 디자인

- 트리거는 기존 입력과 동일한 흰색 배경, 둥근 모서리, 남색 포커스 링을 사용한다.
- 펼친 목록은 흰색 카드, 얇은 테두리, 그림자, 제한된 높이와 스크롤을 사용한다.
- 활성 항목은 옅은 남색, 선택 항목은 브랜드 남색과 체크 표시로 구분한다.
- 화살표 아이콘은 열림 상태에서 회전한다.

## 접근성 및 검증

- 트리거에 `role="combobox"`와 관련 ARIA 상태를 제공하고 목록은 `role="listbox"`를 사용한다.
- 각 항목은 `role="option"`과 `aria-selected`를 제공한다.
- 정렬 유틸리티를 Node 내장 테스트로 검증하고 ESLint와 Vite 빌드로 회귀를 확인한다.
