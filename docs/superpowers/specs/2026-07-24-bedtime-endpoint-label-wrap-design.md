# 취침 시간대 양 끝 라벨 줄바꿈 설계

## 목표

취침 시간대 슬라이더의 1번과 5번 인덱스 설명이 화면 좌우 경계에서 잘리지 않도록 두 줄로 표시한다.

## 동작

- 1번 라벨은 첫 줄에 `10시`, 둘째 줄에 `이전`을 표시한다.
- 5번 라벨은 첫 줄에 `새벽 1시`, 둘째 줄에 `이후`를 표시한다.
- 2번부터 4번까지의 라벨은 기존 한 줄 표시를 유지한다.
- 모든 라벨은 기존 인덱스 위치를 기준으로 가운데 정렬한다.

## 구현

`SurveySleep`의 `indexLabels`에서 1번과 5번 값만 문자열 배열로 전달한다.

```js
{
  1: ['10시', '이전'],
  2: '10시~11시',
  3: '11시~12시',
  4: '12시~1시',
  5: ['새벽 1시', '이후'],
}
```

`Slider`는 라벨 값이 배열이면 각 항목을 별도의 block 요소로 렌더링하고, 문자열이면 기존처럼 그대로 렌더링한다. 인덱스 라벨 영역 높이는 두 줄의 `leading-4`를 수용하도록 `h-8`로 변경한다.

## 검증

- 취침 시간대의 1번과 5번 라벨 값이 지정된 두 줄 배열인지 확인한다.
- `Slider`가 배열 라벨을 줄별 block 요소로 렌더링하는지 확인한다.
- 한 줄 문자열 라벨과 기존 슬라이더 기능이 유지되는지 전체 테스트와 빌드로 확인한다.
